# Newton法による平方根の計算アルゴリズム
## 1. 目的

与えられた正の実数 $a$ に対して、その平方根 $\sqrt{a}$ を数値計算によって近似的に求める。

## 2. 数学的背景

$\sqrt{a}$ を求めることは、以下の非線形方程式の正の解 $x$ を求めることと同値である。
$$f(x) = x^2 - a = 0$$
この解を求めるためにNewton法（Newton-Raphson法）を用いる。関数 $f(x)$ の導関数は $f'(x) = 2x$ である。ある近似値 $x_n$ にける接線が $x$ 軸と交わる点を次の近似値 $x_{n+1}$ とし、これを繰り返すことで真の解に近づけていく。

漸化式は以下のように導出される。
$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$
$$x_{n+1} = x_n - \frac{x_n^2 - a}{2x_n}$$
$$x_{n+1} = \frac{1}{2}\left(x_n + \frac{a}{x_n}\right)$$

## 3. アルゴリズムの手順

1. **入力と例外処理**

   - 計算対象の実数 $a$ を受け取る。
   - $a < 0$ の場合は実数範囲で計算不能なため、エラーとして処理を停止する。
   - $a = 0$ の場合は例外的に $0$ を出力して処理を終了する。

2. **初期化**

   - 近似解の初期値 $x_{new}$ を設定する（実装では入力値 $a$ をそのまま初期値として採用）。
   - 収束判定のための許容誤差 $\epsilon$ を設定する（例：$1.0 \times 10^{-10}$）。
   - 無限ループを防ぐため、最大反復回数 $N_{max}$ を設定する。

3. 反復計算

   - 以下の処理を最大 $N_{max}$ 回繰り返す。現在の近似値を $x_{old}$ として保持する（$x_{old} \leftarrow x_{new}$）。
   - 漸化式を用いて新たな近似値 $x_{new}$ を計算する。
   $$x_{new} = 0.5 \times \left(x_{old} + \frac{a}{x_{old}}\right)$$
   - **収束判定**: 更新前後の差の絶対値が許容誤差未満になったか確認する。
   $$|x_{new} - x_{old}| < \epsilon$$
       この条件を満たせば、十分に値が収束したとみなして反復を終了（break）する。

4. 出力
   - 最終的に得られた $x_{new}$ を結果として出力する。
