# ガウス求積法による定積分アルゴリズム
## 1. 目的
区間 $[-1, +1]$ における4次関数 $f(x) = 1 + x + x^2 + x^3 + x^4$ の定積分を、Gauss求積法を用いて近似的に求め、手計算による厳密解と比較して精度を考察する。

## 2. 数学的背景
### 手計算による厳密解
積分区間が対称であるため、奇数次数の項（$x, x^3$）の積分は $0$ となる性質を利用する。
$$I = \int_{-1}^{+1} (1 + x + x^2 + x^3 + x^4) dx$$
$$I = 2 \int_{0}^{1} (1 + x^2 + x^4) dx$$
$$I = 2 \left[ x + \frac{x^3}{3} + \frac{x^5}{5} \right]_{0}^{1}$$
$$I = 2 \left( 1 + \frac{1}{3} + \frac{1}{5} \right) = 2 \left( \frac{15 + 5 + 3}{15} \right) = \frac{46}{15} \approx 3.06666666667$$

### Gauss求積法
Gauss求積法は、規定されている点（積分点）$\xi_i$ における関数値 $f(\xi_i)$ と、対応する重み $W_i$ の積の和をとることで、積分の近似値 $I_q$ を求める手法である。
$$I_q = \sum_{i=1}^{q} f(\xi_i) W_i$$
$q$ 個の積分点を用いると、$2q-1$ 次までの多項式を誤差なく厳密に積分できるという強力な性質を持つ。本課題の関数は4次式であるため、$2q - 1 \ge 4 \Rightarrow q \ge 2.5$、すなわち $q=3$（3次）以上のGauss求積で手計算の厳密解と完全に一致することが理論的に保証される。

## 3. アルゴリズムの手順
1. 厳密解の算出
   - 手計算で求めた厳密解 $46.0 / 15.0$ を変数に格納する。

2. Gauss求積
   - 関数の定義（サブルーチンの活用）積分対象の関数 $f(x)$ をFortranの function として定義する。
   - Gauss求積を実行するサブルーチンを作成する。引数として次数 $q$ を受け取り、近似値 $I_q$ を返す。
   - サブルーチン内部で $q$ に応じて積分点 $\xi$ と重み $W$ の配列を動的に割り当て、表に記載された数値を代入する。
   - ループを用いて $\sum_{i=1}^{q} f(\xi_i) W_i$ を計算する。

3. 実行と出力
   - $q = 1, 2, 3, 4$ の各次数についてサブルーチンを呼び出し、近似積分値を得る。
   - それぞれの結果について、厳密解との絶対誤差 $|I_q - I_{exact}|$ を計算して出力し、精度の違いを考察可能にする。
